## Tables

- Genre

  • id: pk

  • nom: unique (not null)


- NiveauEtude

  • id: pk

  • nom: unique (not null)


- Employe

  • id: pk

  • matricule: unique (not null)

  • nom

  • prenom

  • dateNaissance: contrainte->CHECK (dateNaissance < CURRENT_DATE - INTERVAL '18 years') (not null)

  • idgenre: fk(Genre.id)

  • idniveauEtude: fk(NiveauEtude.id)


- Specialite

  • id: pk

  • nom: unique (not null)

  • description


- SpecialiteEmploye

  • idemploye: fk(Employe.id)

  • idspecialite: fk(Specialite.id)


## View

- Specialite + SpecialiteEmploye on idemploye: v_specialite_employe
